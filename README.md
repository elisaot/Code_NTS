# Code_NTS

A small collection of scraping utilities for NTS dictionary and files from collecting NRK news broadcasts in NTS. The repository contains a focused scraper in `scrape-SL/` and a couple of data files produced by other processes.

## Contents

- [scrape-SL/](scrape-SL/README.md) — scraper script and project metadata
- [NRK_log.txt](NRK_log.txt) — log file (generated)
- [NRK_urls.txt](NRK_urls.txt) — collected NRK URLs (generated)


See [scrape-SL/README.md](scrape-SL/README.md) for scraper-specific configuration and usage.

## Repository layout

- `scrape-SL/` — the main scraper project (contains `main.py` and `pyproject.toml`)
- `NRK_urls.txt` — output or input list of URLs
- `NRK_log.txt` — runtime log output
